헤더 셀을 구분할 때 3개 이상의 -(하이픈/대시) 기호를 적어줘야 함.
헤더 셀을 구분하면서 :(콜론) 기호로 셀(열/칸) 안에 내용을 정렬할 수 있음.
가장 좌측과 가장 우측에 있는 |(버티컬바) 기호는 생략 가능.
